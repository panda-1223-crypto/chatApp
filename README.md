■ 概要

このプロジェクトは簡易的なチャット風アプリです。

ユーザーの入力を受け取り、バックエンド API を通して

DB から候補データを取得し、アプリ側のロジックで最も適切な応答を選んで表示します。

リアルタイム通信や永続的なユーザー管理は未実装で、

動作確認やアルゴリズム検証を目的としたプロトタイプです。



■　使用技術

・JavaScript

・HTML/CSS

・c♯



■　主な機能

・ユーザー入力を受け取る画面

・バックエンド API へリクエスト（入力を送信）

・DB（テーブル）から候補データを取得する API

・取得データに対してロジック（簡易的なマッチング／スコア算出等）を実行して最適な返答を決定

・返答を画面に表示



■　起動方法

・DBがローカルのため、起動はできません。
